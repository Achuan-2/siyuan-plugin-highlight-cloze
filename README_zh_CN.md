# 高亮挖空


## 🤔背景

在学习过程中，我们经常会遇到一些需要填空的题目、需要记忆的知识点，为了方便记忆，我们可以使用高亮挖空的方式来标记这些知识点和题目，确认是否自己真的记住了这些内容。


## ✨功能

点击顶部的插件按钮，即可隐藏/显示高亮文字
