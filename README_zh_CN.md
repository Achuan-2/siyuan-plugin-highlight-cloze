# 高亮挖空


## 🤔背景

在学习过程中，我们经常会遇到一些需要填空的题目，为了方便记忆，我们可以使用高亮挖空的方式来标记这些题目，这样我们在复习的时候就可以快速找到需要填写的内容。


## ✨功能

支持点击顶部按钮隐藏/显示高亮文字
